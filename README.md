# SkyLine 2D矩形装箱算法

## 项目简介
本项目实现了一种基于**天际线启发式算法（Skyline Heuristic）**的二维矩形装箱解决方案。旨在将多个不同尺寸的矩形高效地放置在一个固定的容器（Container）中，以最大化空间利用率。

## 核心功能
- **天际线算法**：通过维护一条动态变化的“天际线”，优先填充最低、最左的空闲区域，有效减少碎片空间。
- **智能评分系统**：根据矩形与当前天际线及周边环境的匹配度进行打分，选择最优的放置方案。
- **旋转支持**：支持配置是否允许矩形旋转90度，以提高放置成功率。
- **数据驱动**：通过文本文件灵活定义容器尺寸和待放置矩形的规格。

## 技术栈
- **语言**: Java 8+
- **构建工具**: Maven
- **依赖**: Lombok (用于简化实体类代码)

## 快速开始

### 1. 准备环境
确保你的电脑上安装了以下软件：
- JDK 1.8 或更高版本
- Maven 3.x

### 2. 运行项目
在项目根目录下执行以下命令：

```bash
mvn clean compile exec:java -Dexec.mainClass="com.hbz.App"
```

或者在 IDE（如 IntelliJ IDEA, Eclipse）中直接运行 `App.java`。

### 3. 数据格式说明
数据文件位于 `src/main/java/com/hbz/data/data.txt`。

**第一行**：`容器宽度 容器高度 是否允许旋转(1:是, 0:否)`
**后续行**：`矩形宽度 矩形高度`

**示例：**
```text
400 400 1
59 115
26 99
...
```

## 算法原理简述
1. **初始化**：从容器底部生成初始天际线。
2. **迭代**：每次选取最下、最左的天际线段。
3. **评估**：遍历所有未放置的矩形，计算其在当前天际线上的得分（考虑贴合度、高度差等）。
4. **放置**：选择得分最高的矩形进行放置，并更新天际线（分裂或合并）。
5. **结束**：当没有更多矩形可放置或天际线队列为空时结束。

## 输出结果
程序运行后会输出：
- 成功放置的矩形数量及总数量。
- 空间利用率（放置率）。
- 总的放置面积。
